I am Matteo

I design and code Web products

I have been creating with computers since I was 3, using my dad’s work machine to make abstract drawings. And that’s what I still do, more or less: making abstract things with code.

Between then and now, I trained as a communication designer, taught myself how to program computers and made educational games for clients like the BBC.

I'm interested in hacking cultures, critical making and using data to create computational literature (aka bots).

Currently I teach people aged 7 to 60+ how to be creative with code.

You can read about my teaching philosophy at github.com/matteomenapace/tm and follow me at @baddeo.
